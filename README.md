# Restaurant Management System

## Overview

The Restaurant Management System is a simple console-based application developed in C programming. This project was created during the first year of engineering to demonstrate the fundamentals of programming concepts such as functions, loops, conditional statements, arrays, and user interaction.

The system allows users to view a restaurant menu, place orders, calculate the total bill, and choose payment options through an easy-to-use interface.

## Features

* Display restaurant menu items
* Place food orders
* Calculate total bill amount automatically
* Multiple payment options
* User-friendly console interface
* Basic input validation

## Technologies Used

* Programming Language: C
* Platform: Console/Terminal Application
* Compiler: GCC, Turbo C, Dev-C++, CodeBlocks, or any standard C compiler

## Learning Objectives

This project was developed to gain hands-on experience with:

* C programming fundamentals
* Functions and modular programming
* Conditional statements and loops
* User input and output handling
* Basic menu-driven application development

## How to Run

1. Download or clone the project.
2. Open the source code in any C compiler or IDE.
3. Compile the program:

   ```bash
   gcc restaurant_management_system.c -o restaurant
   ```
4. Run the executable:

   ```bash
   ./restaurant
   ```

   (For Windows)

   ```bash
   restaurant.exe
   ```

## Project Workflow

1. Display the restaurant menu.
2. User selects food items.
3. Quantity and order details are processed.
4. Total bill is calculated.
5. User chooses a payment method.
6. Bill and payment details are displayed.
